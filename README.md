# Практичне завдання №2 з ОС

```diff
+ Завдання 1.
+   1) Сформувати масив з n елементів за допомогою датчика випадкових чисел (n задається користувачем з клавіатури).
+   2) Роздрукувати отриманий масив.
+   3) Виконати видалення зазначених елементів з масиву (Максимальний елемент).
+   4) Вивести отриманий результат.
+   5) Виконати додавання зазначених елементів в масив (К елементів в початок масива).
+   6) Вивести отриманий результат.
+   7) Виконати перестановку елементів у масиві (Перевернути масив).
+   8) Вивести отриманий результат.
+   9) Виконати пошук вказаних (перший парний) в масиві елементів і підрахувати кількість порівнянь, необхідних для пошуку потрібного елемента.
+   10) Вивести отриманий результат.
+   11) Виконати сортування масиву зазначеним методом.
+   12) Вивести отриманий результат.
+   13) Виконати пошук вказаних (перший парний) елементів у відсортованому масиві і підрахувати кількість порівнянь, необхідних для пошуку потрібного елемента.
+   14) Вивести отриманий результат.

+ Завдання 2. Дано масив A (N). Знайти: max(a2, a4, ..., a2k) + min(a1, a3, a2k+1).

+ Завдання 3. Обчислити добуток від'ємних чисел головної діагоналі матриці.

+ Завдання 4.
+   1) Сформувати динамічний одновимірний масив, заповнити його випадковими числами і вивести на друк.
+   2) Виконати вказану у варіанті завдання дію (Видалити перший парний елемент) і вивести отриманий масив на друк.
+   3) Сформувати динамічний двовимірний масив, заповнити його випадковими числами і вивести на друк.
+   4) Виконати вказану у варіанті завдання дію (Додати рядок із заданим номером) і вивести отриманий масив на друк.

+ Завдання 5. Ввести з клавіатури рядок символів і обробити її у відповідності зі своїм варіантом, використовуючи функції
+   1) Видалити всі голосні літери з рядка.
+   2) Задано послідовність стрингів різної довжини. Отримати номери  всіх рядків, що містять як фрагмент перший рядок. В результаті отримати послідовності стрингів з виключеними пробілами та знайденими фрагментами першого рядка.
```
